# Git

## Commits

- Never add `Co-Authored-By: Claude` or any Claude / AI attribution to commit messages.
- Keep commit messages concise and descriptive.

## Branches

- Use simple, descriptive branch names (e.g. `nyse-holidays`, `fix-nav-change`).
- Do not prefix branch names with conventional-commit types like `feat/`, `fix/`, `chore/`, etc.
